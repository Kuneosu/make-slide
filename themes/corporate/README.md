# Corporate

## Design Philosophy / 디자인 철학

**EN:** A professional, structured theme built for business presentations and corporate reporting. Navy and gold accents convey authority and trust, while clean grid layouts and restrained typography keep the focus on data and key messages. Designed for boardrooms, investor meetings, and quarterly reviews.

**KO:** 비즈니스 프레젠테이션과 기업 보고서를 위한 전문적이고 구조화된 테마입니다. 네이비와 골드 액센트가 신뢰감과 권위를 전달하며, 깔끔한 그리드 레이아웃과 절제된 타이포그래피가 데이터와 핵심 메시지에 집중할 수 있게 합니다. 회의실, 투자자 미팅, 분기 보고에 적합합니다.

## Slide Types / 슬라이드 타입

`reference.html` includes examples of all supported slide types:

1. **Title** — Large centered headline with subtitle, date, and tag
2. **Agenda/TOC** — Numbered topic list with gold index numbers
3. **Content** — Body text with bullet points and gold highlights
4. **Quote** — Large quotation with gold accent mark and attribution
5. **Comparison** — Before/After two-column layout with status indicators
6. **Flow/Steps** — Horizontal process diagram with 4 numbered phases
7. **Cards** — 3 feature cards in a grid with navy top borders
8. **Data/Chart** — CSS bar chart with navy bars and mono-spaced values
9. **Code Block** — Syntax-highlighted code with Prism.js and file header
10. **Image Placeholder** — Dashed border placeholder area on light surface
11. **Closing** — Thank you slide with contact links and gold underline
