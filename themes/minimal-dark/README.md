# Minimal Dark

## Design Philosophy / 디자인 철학

**EN:** A clean, modern dark theme designed for tech conferences and developer presentations. The dark background eliminates visual noise, letting your content take center stage. Minimal decoration, maximum clarity.

**KO:** 테크 컨퍼런스와 개발자 발표를 위한 깔끔하고 모던한 다크 테마입니다. 어두운 배경이 시각적 잡음을 제거하고, 콘텐츠에 집중할 수 있게 합니다. 최소한의 장식, 최대한의 명확성.

## Use Cases / 적합한 용도

**EN:**
- Tech conferences and meetups
- Developer tool demos
- Open source project presentations
- Technical architecture overviews
- Startup pitch decks (tech-focused)
- Late-night / dim-room presentations

**KO:**
- 테크 컨퍼런스 및 밋업

## Slide Types / 슬라이드 타입

`reference.html` includes examples of all supported slide types:

1. **Title** — Large centered headline with subtitle
2. **Agenda/TOC** — Numbered topic list
3. **Content** — Body text with highlights
4. **Quote** — Large quotation with attribution
5. **Comparison** — Before/After two-column layout
6. **Flow/Steps** — Horizontal process diagram
7. **Cards** — Feature cards grid
8. **Data/Chart** — CSS bar chart
9. **Code Block** — Syntax-highlighted code (Prism.js)
10. **Image Placeholder** — Placeholder for visual content
11. **Closing** — Thank you / end slide
