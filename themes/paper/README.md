# Paper

## Design Philosophy / 디자인 철학

**EN:** A warm, serif-driven theme inspired by the tactile quality of kraft paper and printed books. Off-white backgrounds, Noto Serif typography, and earthy terracotta accents create an academic, hand-crafted atmosphere. The design evokes the comfort of a well-worn notebook — approachable, readable, and quietly elegant.

**KO:** 크래프트 종이와 인쇄된 책의 촉감에서 영감을 받은 따뜻한 세리프 기반 테마입니다. 오프화이트 배경, Noto Serif 타이포그래피, 테라코타 계열 악센트 컬러가 학술적이고 수공예적인 분위기를 만들어냅니다. 잘 사용된 노트북의 편안함을 연상시키며, 접근하기 쉽고, 읽기 편하며, 조용한 우아함을 갖추고 있습니다.

## Slide Types / 슬라이드 타입

`reference.html` includes examples of all supported slide types:

1. **Title** — Large centered serif headline with subtitle, date, and tag label
2. **Agenda/TOC** — Numbered topic list with card-style items
3. **Content** — Body text with serif typography and terracotta highlights
4. **Quote** — Literary quotation mark with italic text and attribution
5. **Comparison** — Before/After two-column layout with accent border
6. **Flow/Steps** — Horizontal four-step process with numbered labels and arrows
7. **Cards** — Three feature cards with paper-like shadow
8. **Data/Chart** — CSS bar chart with terracotta gradient values
9. **Code Block** — Syntax-highlighted code with sepia-tinted background and file header
10. **Image Placeholder** — Dashed border placeholder area
11. **Closing** — Thank you slide with contact links
