04

코드 → 바로 테스트 vs
코드 → TC → 테스트

일반적 접근

코드 → AI → 테스트 생성

구현 사실만 확인하는
형식적인 테스트가 나옴

더 나은 접근

코드 분석 → TC 문서 생성 → 검토 → 테스트 코드

TC를 먼저 만들고 검토하면
훨씬 정밀한 테스트가 나옴

중간 단계(TC)를 거치는 것만으로 테스트 품질이 크게 달라진다
